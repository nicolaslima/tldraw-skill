tldraw-skill

手绘 .tldr 文件,带视觉自检循环、迭代评审、6 种图表预设。基于 @kitschpatrol/tldraw-cli。

/plugin marketplace add Agents365-ai/365-skills
GitHub ClawHub SkillsMP Discord

来自一条自然语言提示词 — 完整提示词见 README。

为什么选这个技能

白板草图与商务正式图的规则不一样。本技能内置布局、配色、自检规则,让 tldraw 输出第一次就对。

📝

6 种图表预设

架构图、流程图、时序图、ML / 深度学习、ERD、UML —— 每种都有预设的形状词汇与布局规范。

🔍

视觉自检

技能读取导出的 PNG,自动修复 6 类问题(重叠、文字截断、缺失箭头),修好再给你看。

🔄

迭代评审循环

收集反馈、精准编辑 JSON、重新导出,直到满意。5 轮安全阀防止无限循环。

📐

复杂度自适应布局

间距随节点数自动放大,防止重叠。路由走廊和枢纽节点居中策略内置。

🎨

语义配色

10 色调色板按语义映射(蓝=服务、绿=DB、紫=认证),再也不会每次随机配色。

⚡

零浏览器自动化

tldraw-cli 在哪里有 Node 就在哪里跑。无需 Chromium、Firefox、Playwright —— macOS / Linux / Windows 安装一致。

安装

选择你的平台,只要 10 秒。

# Plugin marketplace (recommended)
/plugin marketplace add Agents365-ai/365-skills
/plugin install tldraw

# Or any agent
npx skills add Agents365-ai/365-skills -g

# Manual install
git clone https://github.com/Agents365-ai/tldraw-skill.git ~/.claude/skills/tldraw-skill
# Global
git clone https://github.com/Agents365-ai/tldraw-skill.git ~/.config/opencode/skills/tldraw-skill

# Project
git clone https://github.com/Agents365-ai/tldraw-skill.git .opencode/skills/tldraw-skill
# Global
git clone https://github.com/Agents365-ai/tldraw-skill.git ~/.openclaw/skills/tldraw-skill
# Via SkillsMP CLI

# User-level
